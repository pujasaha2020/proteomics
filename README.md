# proteomics

This repository documents the project and saves scripts for data construction and data analysis.

## ⚙️ Installation

Clone the repository:

```shell
git clone https://github.com/AdrienSpecht/plasmatime
```

Run setup.sh:

```shell
cd proteomics
./setup.sh
```
